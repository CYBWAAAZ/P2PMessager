# 📡 P2P Messager (P2PMessager)

CLI-чат peer-to-peer на Python 🐍.  
Позволяет напрямую обмениваться сообщениями между устройствами в локальной сети без посредника.

---

## 🎯 Цель проекта

Проект создан в **образовательных целях** и помогает разобраться в:

- Основах компьютерных сетей (TCP-соединения)
- Архитектуре клиент–сервер vs peer-to-peer
- Асинхронном программировании на Python (asyncio)
- Базовом анализе сетевого трафика

---

## 🎓 Образовательные сценарии

P2PMessager можно использовать вместе с инструментами:

- Wireshark
- tcpdump
- FTK Imager

для анализа:

- TCP-соединений
- различий между зашифрованным и открытым трафиком
