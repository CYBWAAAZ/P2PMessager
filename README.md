# 📡 P2P Messager (P2PMessager)

CLI-based peer-to-peer chat application written in Python 🐍.  
Allows direct communication between devices in local networks without central servers.

---

## 🎯 Purpose

This project is created for **educational purposes**, focusing on:

- Fundamentals of computer networks (TCP communication)
- Client-server vs peer-to-peer architecture
- Asynchronous programming in Python (asyncio)
- Basic network traffic analysis

---

## 🎓 Educational Labs

P2PMessager can be used together with tools like:

- Wireshark
- tcpdump
- FTK Imager

to analyze:

- TCP connections
- message framing
- encrypted vs plaintext traffic

---

## ⚙️ Versions

### 📌 P2PMessager 
- TCP-based communication
- CLI chat interface
- Plaintext messages (no encryption)
- Ideal for network traffic analysis

---

### 🔐 P2PMessager Encrypted Edition
- TCP-based communication
- The ability to use encryption

---

### 🛡 P2PMessager Security Edition (WIP)
- Default encrypted communication
